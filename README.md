# Головоломка со спичками

Windows Forms игра на C++/CLI: игрок переносит спички, чтобы из неверного равенства получить указанную правильную цель.

## Запуск

1. Установите Visual Studio 2022 с компонентами `Desktop development with C++` и `C++/CLI support`.
2. Запустите `Запустить_игру.bat` или `run.bat`.
3. Скрипт соберет `Release|x64` и откроет `bin\Release\MatchstickPuzzle.exe`.

Также можно открыть `MatchstickPuzzle.sln` в Visual Studio и собрать проект вручную.

## Что есть в игре

- режимы: кампания, игра на время, свободный режим;
- сложности: легко, средне, сложно;
- перенос спичек кликом: сначала активная спичка, затем пустое место;
- лимит ходов, подсказки, счет и таймер.
